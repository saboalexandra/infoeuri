(E1520, Propilen glicol, Propan 1,2-diol, Metil diol)

STABILIZATOR UMECTANT AGENT ANTIAGLOMERANT AGENT ANTISPUMANT EMULGATOR AGENT DE INGROSARE

Periculozitate:

Face parte din categoria "E-uri periculoase".

Nu este permis in alimentele destinate sugarilor si copiilor mici.

Reactii adverse:

  • Stari depresive Produce stari depresive ale sistemului nervos central.
  • Neurotoxic
  • Cardiotoxic
  • Dermatite In contact cu pielea produce dermatite.
