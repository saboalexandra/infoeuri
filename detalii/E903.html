(E903, Ceara de Carnauba)

AGENT DE GLAZURARE EMULGATOR AGENT ANTIAGLOMERANT AGENT DE INGROSARE STABILIZATOR

Periculozitate:

Face parte din categoria "E-uri suspecte".

Nu este permis in alimentele destinate sugarilor si copiilor mici.

Reactii adverse:

  • Eczeme cutanate Ocazional cauzeaza reactii alergice (eczeme cutanate).
  • Carcinogen Este suspectat ca produce carcinom.
