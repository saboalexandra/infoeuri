(E406, Agar-agar, Geloza)

EMULGATOR STABILIZATOR AGENT DE INGROSARE AGENT DE AGLOMERARE

Periculozitate:

Face parte din categoria "E-uri suspecte".

Nu este permis in alimentele destinate sugarilor si copiilor mici.

Reactii adverse:

  • Poate diminua asimilarea substantelor minerale si altor nutrienti in intestin
  • Laxativ Daca este consumat in cantitati mari.
